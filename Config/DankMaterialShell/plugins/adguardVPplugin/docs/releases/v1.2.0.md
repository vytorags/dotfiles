# AdGuard VPN Plugin v1.2.0

This release focuses on connection resilience, location workflow improvements, and better operational diagnostics.

## Highlights

- Added favorites and search/filter for faster location selection.
- Added auto-connect on startup and auto-reconnect on unexpected drops.
- Added tunnel log opening and last-command diagnostics directly in the UI.
- Improved CLI parsing robustness by extracting dedicated parser utilities.
- Added quality and contribution workflow improvements (i18n parity check, CI quality pipeline, issue/PR templates).

## Compatibility

- Requires DankMaterialShell `>= 1.4.0`.
- Requires `adguardvpn-cli` available in PATH (or configured binary path).
