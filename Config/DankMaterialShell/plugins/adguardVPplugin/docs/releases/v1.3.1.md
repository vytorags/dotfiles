# AdGuard VPN Plugin v1.3.1

This hotfix release improves account detection reliability and diagnostics in environments where `adguardvpn-cli license` can be slow.

## Highlights

- Increased reliability of account detection by hardening the license refresh flow.
- Added a safer timeout strategy for slow `license` calls plus an in-flight watchdog.
- Expanded license parsing patterns for alternate output formats.
- Preserved valid account metadata when transient/partial output occurs.
- Ensured diagnostics always show the latest attempted command, including preflight failures.
- Fixed popup compatibility by removing unsupported `selectByMouse` usage in the DNS field.
- Updated PT-BR copy for the multiple-default-route warning to "rotas padrão".

## Compatibility

- Requires DankMaterialShell `>= 1.4.0`.
- Requires `adguardvpn-cli` available in PATH (or configured binary path).
