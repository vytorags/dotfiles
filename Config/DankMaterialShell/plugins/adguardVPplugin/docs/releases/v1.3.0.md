# AdGuard VPN Plugin v1.3.0

This release introduces a major UI refresh, runtime connection hardening, and a broad multilingual expansion.

## Highlights

- Full redesign of the popout and settings surfaces with clearer hierarchy and faster action flow.
- Added runtime preflight checks before connect to handle stale socket and busy runtime states.
- Added explicit guard for multi-default-route scenarios in TUN mode.
- Expanded localization coverage to 22 locales with a safe fallback strategy.
- Upgraded i18n validation tooling to check all locale files and report fallback coverage.

## Compatibility

- Requires DankMaterialShell `>= 1.4.0`.
- Requires `adguardvpn-cli` available in PATH (or configured binary path).
